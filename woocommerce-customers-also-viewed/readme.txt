== Description ==
Show a group of products that were viewed by other customers that also viewed the current product, ie. Customers who viewed this also viewed that.

== Installation ==
1. Install the plugin (Requires Woocommerce)
2. Go to the `Woocommerce -> Customers Also Viewed` menu to manage the plugin options.
